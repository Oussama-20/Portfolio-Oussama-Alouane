Projet Student Management System (CRUD)

Application web de gestion des étudiants

Objectif du projet

Développer une application web permettant de gérer les informations des étudiants (ajout, lecture, mise à jour, suppression) à l'aide des opérations CRUD. L’interface est simple, responsive et adaptée aux besoins scolaires.

Fonctionnalités principales

  • Ajouter un étudiant

    Ajouter un étudiant via un formulaire complet et validé.
  • Afficher les données — Voir la liste des étudiants avec leurs informations.
  • Modifier — Mettre à jour les informations existantes.
  • Supprimer — Retirer un étudiant avec confirmation.
  • Recherche — Filtrer les étudiants par nom, prénom ou adresse.
  • Menu — Accès aux sections : Students, Teachers, Courses, Enrollment, Payment.

Technologies utilisées

  • HTML5
  • CSS3
  • Bootstrap
  • Laravel
  • MySQL
